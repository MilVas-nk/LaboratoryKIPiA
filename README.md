# Лаборатория КИПиА

Приложение LaboratoryKIPiA - это Android-приложение на языке Kotlin, предназначенное для проверки возможности поверки средств измерения в рамках аккредитации лаборатории.

## Назначение

Приложение позволяет пользователю ввести значение ВПИ (верхнего предела измерения) и погрешность/класс точности прибора, а затем проверить, попадает ли прибор в диапазоны аккредитации лаборатории по заданным параметрам. Это позволяет оперативно определить, может ли лаборатория выполнить поверку конкретного средства измерения.

## Технологии и зависимости

- Язык программирования: Kotlin
- Платформа: Android (API 24+)
- Сборка проекта: Gradle (Kotlin DSL)
- Поддержка Material Design через библиотеку com.google.android.material
- Совместимость с androidx библиотеками

### Зависимости

- `androidx.core.ktx` версии 1.10.1
- `androidx.appcompat` версии 1.6.1
- `com.google.android.material` версии 1.10.0
- Тестирование: JUnit 4.13.2, AndroidX JUnit 1.1.5, Espresso 3.5.1

## Функциональные возможности

- Позволяет пользователю ввести значение ВПИ (верхнего предела измерения) и погрешность/класс точности прибора
- Предоставляет выбор между типами средств измерения
- Проверяет, попадает ли прибор в диапазоны аккредитации лаборатории по заданным параметрам
- Отображает результат проверки (можно/нельзя поверить) с пояснениями

## Примеры использования

### Пример 1: Проверка датчика давления

1. Выберите "Датчик давления" в переключателе типа прибора
2. Введите ВПИ: 2.5 (МПа)
3. Введите погрешность: 0.5 (%)
4. Нажмите кнопку "Проверить"
5. Приложение покажет, можно ли поверить данный прибор в лаборатории

### Пример 2: Проверка манометра

1. Выберите "Манометр" в переключателе типа прибора
2. Введите ВПИ: 10 (МПа)
3. Введите класс точности: 1.0
4. Нажмите кнопку "Проверить"
5. Приложение покажет результат проверки с пояснением
