Наименоевание приложения: Лаборатория КИПиА.
Назначение: Проверка соответствия средств измерения утвержденной области аккредитации лаборатории.
Описание: В приложение можно проверить соответствие выбранного средства измерения утвержденной области аккредитации путем выбора типа средства измерения, внесением в соответствующие поля верхнего предела измерения и погрешности.
